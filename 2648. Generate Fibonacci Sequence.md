# Fibonacci Sequence Generator

A JavaScript/TypeScript generator function that yields the Fibonacci sequence indefinitely.

## Problem Description

Write a generator function that returns a generator object which yields the Fibonacci sequence.

The Fibonacci sequence is defined by the relation: `Xn = Xn-1 + Xn-2`

The first few numbers of the series are: `0, 1, 1, 2, 3, 5, 8, 13, ...`

## Solution

### JavaScript Implementation

```javascript
/**
 * @return {Generator<number>}
 */
var fibGenerator = function*() {
    let a = 0, b = 1;
    
    // Yield the first number (0)
    yield a;
    
    // Infinite loop to keep generating Fibonacci numbers
    while (true) {
        // Yield the next number (1, then 1, then 2, then 3, etc.)
        yield b;
        
        // Calculate the next Fibonacci number
        [a, b] = [b, a + b];
    }
};
```

### TypeScript Implementation

```typescript
function* fibGenerator(): Generator<number, any, number> {
    let a = 0, b = 1;
    
    yield a;
    
    while (true) {
        yield b;
        [a, b] = [b, a + b];
    }
};
```

## How It Works

1. **Initialization**: Start with `a = 0` and `b = 1`, the first two Fibonacci numbers
2. **First Yield**: Immediately yield `0` (the value of `a`)
3. **Infinite Loop**: 
   - Yield the current value of `b`
   - Update `a` and `b` using array destructuring:
     - `a` becomes the old `b`
     - `b` becomes `a + b` (sum of previous two numbers)

## Sequence Generation

The generator produces the following sequence:
- First call: `0`
- Second call: `1`
- Third call: `1`
- Fourth call: `2`
- Fifth call: `3`
- Sixth call: `5`
- Seventh call: `8`
- And so on...

## Usage Examples

### Example 1: First 5 numbers
```javascript
const gen = fibGenerator();
console.log(gen.next().value); // 0
console.log(gen.next().value); // 1
console.log(gen.next().value); // 1
console.log(gen.next().value); // 2
console.log(gen.next().value); // 3
```

### Example 2: First 8 numbers
```javascript
const gen = fibGenerator();
const result = [];
for (let i = 0; i < 8; i++) {
    result.push(gen.next().value);
}
console.log(result); // [0, 1, 1, 2, 3, 5, 8, 13]
```

## Alternative Implementation

Without using array destructuring:

```javascript
var fibGenerator = function*() {
    let a = 0, b = 1;
    
    yield a;
    
    while (true) {
        yield b;
        const next = a + b;
        a = b;
        b = next;
    }
};
```

## Features

- ✅ Generates Fibonacci sequence indefinitely
- ✅ Memory efficient (only stores last two numbers)
- ✅ Works with any number of calls (0 to infinity)
- ✅ Compatible with both JavaScript and TypeScript
- ✅ Follows generator function conventions

## Constraints

- `0 <= callCount <= 50` (as per problem constraints, but generator works beyond this)

## Time & Space Complexity

- **Time Complexity**: O(1) per call
- **Space Complexity**: O(1) (constant space, only stores two variables)
